Тут будет конкретика по файлу


Для запуска проекта с начала нужно запустить .py файл для создания Базы дынных, требуется ввести в консоль:
pip install Flask
pip install Flask-SQLAlchemy
pip install flask_cors 
Или установить одноименные библиотеки через интерфейс

После чего запустить RUN_THIS.py и открыть файл index.html через браузер

Для командной работы используется Git-репозиторий ---, или скидывать в группу

Для работы с Git можно загрузить оф. инсталятор https://git-scm.com/install/

Как пользоваться Git:
Команды для загрузки проекта в репозиторий:
1. git config user.name "Имя пользователя" (делается 1 раз)
2. git status
3. git add -A
4. git commit -m "Краткое описание"
5. git push -u origin master

.gitignore - файл в котором игнорируются какие-либо файлы проекта (к примеру Базы Данных или локальные установки). Для того, что бы проигнорировать файл, следует прописать его название и расширение в .gitignore и поставить "/" в конце, для игнорирования папки расширение не нужно
команды:
git status - покажет, какие файлы были изменены, добавлены или удалены.
git add -A - добавить все файлы (или git add --all)
